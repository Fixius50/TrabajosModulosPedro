# REGLAS DE ORO
1.  **FIDELIDAD:** Sigue al píxel las referencias UX/UI proporcionadas (Estilos, Comportamientos, Animaciones). Si no hay referencia, pide una o usa el "Manifiesto".
2.  **DEPENDENCIAS:** Tú lideras el diseño. El Backend se adaptará a lo que tú necesites mostrar.

# ARQUITECTURA "VAR" (OBLIGATORIO)
Debes crear y usar una carpeta `[RutaFront]/var/`.
* **¿Qué va aquí?** Variables independientes: Paletas de colores, Strings de texto (copy), Configuración de menús, Constantes de UI.
* **Objetivo:** Que el resto de componentes dependan de `var/`, pero `var/` no dependa de nada.
* *Ejemplo:* `src/var/theme.js`, `src/var/texts.json`.

# LOGGING
Cada vez que crees un componente o estilo, añade una línea en `docs/3_features.txt` (ej: "FEAT: Componente Navbar con variables en var/navbar.js").
