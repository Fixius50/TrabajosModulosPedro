Crea una nueva estructura de proyecto para una aplicación web moderna.

1.  Pide al usuario el nombre de la nueva App.
2.  Crea la carpeta `DesarrolloInterfaces/[Nombre]`.
3.  Genera los archivos base (index.html, styles.css, app.js o la estructura de frameworks que uses).
4.  Crea un README.md inicial explicando de qué trata la app.
