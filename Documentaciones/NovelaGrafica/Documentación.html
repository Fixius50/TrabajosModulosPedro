02. Componentes Críticos

Neural Map (Árbol del Destino)

GRAPH_VISUALIZATION

RECURSIVE

Sistema de visualización de trama no lineal. Renderiza nodos como un "Circuito Raíz" Neon.

  • Feature Killer: "Rewind Logic". Permite resetear el estado de la historia a cualquier nodo visitado.
  • Layout: Sistema de autolayout basado en grids para estabilidad visual.
  • Estado: Distinción visual entre Nodos Actuales (Pulso), Visitados y Bloqueados.

Visual Novel Canvas

CORE_ENGINE

El lienzo principal de lectura.

  • Typewriter Effect: Renderizado de texto letra a letra con cursor parpadeante.
  • Input Blocking: Previene decisiones accidentales ocultando opciones hasta finalizar el texto.
  • Overlay System: Interfaz flotante que no obstruye el arte de fondo.
