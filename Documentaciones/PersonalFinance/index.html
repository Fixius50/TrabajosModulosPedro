II. El Arsenal Arcano (MCP)

Los MCP (Model Context Protocol) son las herramientas arcanas que permitieron a la IA interactuar con el mundo físico y el código.

[S] StitchMCP

Invocador de interfaces. Permitió generar pantallas enteras desde descripciones rúnicas (texto).

mcp_StitchMCP_generate_screen_from_text

[B] Supabase MCP

El puente hacia el reino de la persistencia. Gestionó tablas, triggers y RLS directamente.

mcp_supabase-mcp-server_execute_sql

[F] Filesystem MCP

El escriba digital. Manipuló físicamente los archivos del proyecto y la estructura de carpetas.

mcp_filesystem_write_file

[N] NotebookLM MCP

El bibliotecario arcano. Proporcionó el contexto de investigación basado en documentos previos.

mcp_notebooklm_ask_question

API Integrations

Además de los MCPs, FinFlow consume servicios externos para su sabiduría:

  • ◈ CoinGecko API: Para el valor de activos en tiempo real.
  • ◈ Supabase Auth/Storage: Gestión de identidades y bóveda de archivos.
  • ◈ Sentry: Ojo que todo lo ve para la detección de errores en el reino.

◈ Forjado del Nexo de Datos

"Usa el MCP de Supabase para configurar la infraestructura. Crea una tabla de transacciones que soporte adjuntos multimedia. Lo más crítico: implementa un sistema de Hogares (Households) donde múltiples usuarios compartan gastos. Escribe un Trigger en PostgreSQL que asegure la integridad atómica: al crear un hogar, el user_id del creador debe insertarse inmediatamente como admin en la tabla de miembros. No olvides las políticas RLS para que ningún mago vea el oro de otro."

◈ Alquimia Visual con Nano Banana

"Necesito iconos personalizados que respiren magia. Utiliza Nano Banana para generar 12 glifos en formato PNG. Cada glifo debe representar una categoría financiera: una runa de fuego para 'Gastos Urgentes', una runa de agua para 'Ocio' y una runa de tierra para 'Ahorros'. Integra estos iconos en el CategoryPicker y asegúrate de que tengan un filtro de drop-shadow dorado que palpite suavemente al ser seleccionados."

◈ Transmutación a Materia (Android)

"Es hora de que la web trascienda al plano físico. Configura Capacitor en el proyecto actual. Ajusta el capacitor.config.ts para que el nombre de la app sea 'FinFlow' y el paquete 'com.roberto.finflow'. Analiza el reporte de linting para corregir cualquier 'Unexpected any' que pueda romper la compilación de Android. Finalmente, genera los comandos necesarios para sincronizar la carpeta dist con el proyecto de Android Studio."

IV. Forjado del APK

Sigue los pasos rituales para materializar el grimorio en un dispositivo Android.

01. Purificación del Código

npm run build

02. Apertura del Portal Nativo

npx cap add android

03. Alineación de Dimensiones

npx cap sync

04. El Taller de Google

npx cap open android

Encantamiento Final (Build)

Dentro de Android Studio, ejecuta el encantamiento de compilación:

Build > Build Bundle(s) / APK(s) > Build APK(s)

Destino: app/build/outputs/apk/debug/app-debug.apk
