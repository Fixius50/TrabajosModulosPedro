II. El Sendero de la IA

Génesis de la Arquitectura

El primer rito para establecer la base sólida del grimorio.

"Necesito crear una SPA profesional con React y TypeScript. El objetivo es una gestión de finanzas personales pero con una estética de Fantasía Oscura (Dark Fantasy). Configura el proyecto para usar Supabase como backend, pero asegúrate de que toda la UI use Glassmorphism premium. Crea una carpeta Docs/ donde gestiones el roadmap, las reglas de negocio y los diagramas de arquitectura. Quiero que seas proactivo y sugieras patrones de diseño escalables."

Alquimia Rúnica e Iconografía

Invocación de elementos visuales únicos mediante Nano Banana.

"Actúa como un Senior UI Designer. Para el Dashboard de FinFlow, no quiero iconos estándar de Material UI. Usa la herramienta Nano Banana para generar una serie de glifos rúnicos dorados sobre fondos oscuros que representen: Comida (Runa de Nutrición), Alquiler (Runa del Refugio), y Salario (Runa de la Abundancia). Una vez generados, implementa un componente IconRing que use estos assets con filtros de brillo dinámicos al hacer hover."

Atuendo Multiprocesador

Lógica de base de datos para la gestión colaborativa.

"Tengo que implementar la funcionalidad de 'Hogares'. Escribe un script SQL de migración para Supabase que cree la tabla households y una tabla intermedia de miembros. Lo más importante: crea un Trigger en PostgreSQL que se dispare después de insertar un hogar; este trigger debe insertar automáticamente al usuario creador en la tabla de miembros con el rol de 'ADMIN'. Realiza esto de forma que sea una transacción atómica para evitar desincronizaciones."

Transmutación Móvil

Reacondicionamiento para el mundo táctil nativo.

"La app debe ser Mobile First. Analiza todos los archivos CSS y componentes del directorio src/. Reemplaza cualquier unidad px hardcoded por unidades rem y vh/vw para que el escalado sea perfecto en pantallas de alta densidad de móviles. Después, configura Capacitor en el proyecto, ajustando el appId a 'com.roberto.finflow' y habilita el esquema HTTPS para las peticiones a la API en Android."

III. Forjado del APK

Sigue estos pasos rituales para transformar el código web en una aplicación nativa instalable en Android.

01

Substitución de Energía (Build)

Compila todo el código TypeScript y React en archivos estáticos optimizados.

npm run build
02

Portal de Dimensión (Android Add)

Crea el contenedor nativo de Android dentro de tu proyecto.

npx cap add android
03

Sincronía de Almas (Sync)

Copia los archivos construidos de la carpeta dist al proyecto de Android.

npx cap sync
04

Apertura del Taller (Studio)

Lanza el entorno de compilación de Google para el paso final.

npx cap open android
05

Extracción de la Esencia (Build APK)

Dentro de Android Studio, ve al menú superior:

Build > Build Bundle(s) / APK(s) > Build APK(s)

El archivo resultante se encontrará en:
app/build/outputs/apk/debug/app-debug.apk
